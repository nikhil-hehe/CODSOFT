# CODSOFT - Machine Learning Internship

This repository contains my project submissions for the Machine Learning Virtual Internship at **CodSoft** (Duration: June 25, 2025 – July 25, 2025). 

---

## 🚀 Projects & Datasets

### 1. Movie Genre Classification
* **Objective:** Predict the genre of a movie based on its textual plot summary.
* **Algorithm:** Multinomial Naive Bayes paired with TF-IDF Vectorization.
* **Dataset Used:** [IMDb Genre Classification Dataset](https://www.kaggle.com/datasets/hijest/genre-classification-dataset-imdb)

### 2. Credit Card Fraud Detection
* **Objective:** Build a classifier to identify fraudulent transactions from historical records.
* **Algorithm:** Random Forest Classifier with explicit class balancing.
* **Dataset Used:** [Credit Card Fraud Detection Dataset](https://www.kaggle.com/datasets/kartik2112/fraud-detection)

### 3. Spam SMS Detection
* **Objective:** Classify SMS mobile text messages as either `spam` or `ham` (legitimate).
* **Algorithm:** Multinomial Naive Bayes paired with TF-IDF Vectorization.
* **Dataset Used:** [SMS Spam Collection Dataset](https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset)
